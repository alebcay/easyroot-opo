easyroot-opo
============

Easy root for OnePlus one, works on linux and mac

Start-new.sh = Is for the new Bootloader (will wipe all data!).
Start.sh = Is for the old Bootloader.

Start-new:
-Open the bootloader
-Flash new recovery (TWRP)
-Flash superuser

Start:
-Fastboot boot recovery
-Flash superuser
